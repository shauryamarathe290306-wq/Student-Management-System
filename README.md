# 🎓 Student Management System (Python)

A simple yet powerful **console-based Student Management System** built using Python.  
It allows users to manage student records with features like add, update, delete, search, analytics, and export to Excel.

---

## ✨ Features

- ➕ Add new students
- 👀 View all students
- 🔍 Search student by roll number
- ✏️ Update student details
- ❌ Delete student records
- 📊 View analytics (average marks, topper, total students)
- 💾 Persistent data storage using JSON
- 📁 Export student data to Excel

---

## 🛠️ Tech Stack

- Python 🐍
- JSON (for data storage)
- Pandas (for Excel export)
- OpenPyXL (Excel engine)
